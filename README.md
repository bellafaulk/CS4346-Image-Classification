# CS4346-Image-Classification

## Setup Instructions
1. Clone the repository:
```git clone https://github.com/bellafaulk/CS4346-Image-Classification```

2. Move to the project folder:
```cd CS4346-Image-Classification```

3. Check branches:
```git branch -a```

## Branches (IMPORTANT)

Each team member works on a separate branch:

- Perceptron (Bella)
- Naive-bayes (Maria)
- Pipeline: data, features, and evaluation (Adami)

To switch to your branch, use:
```git checkout perceptron```
